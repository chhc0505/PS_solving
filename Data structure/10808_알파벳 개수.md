##Problem

<https://www.acmicpc.net/problem/10808>

Data structure - array

## Comment

* ASCII 변환을 이용해서 해결할 수 있는 문제
* ASCII 코드란 ?
    컴퓨터는 정보를 저장할 때 이진수로 저장하기 때문에 문자를 저장하기 위해 문자를 숫자에 매핑시킨 것
    A : 65 , a : 97

## Code
'''c++
#include<bits/stdc++.h>

using namespace std;

int char_cnt[26];
int main () {
    
    ios::sync_with_stdio(0); cin.tie(0);
    string s; cin >> s;
    
    for (auto c: s) 
        char_cnt[c-'a'] += 1;
    
    for (int i=0; i<26; i++) {
        cout << char_cnt[i] << ' ';
    }
    cout << '\n';
    
    return 0;
}
'''
